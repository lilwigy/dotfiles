# Requirement

- Neovim nightly (**it a must!!!**)
- fzf
- rg
- fd
- gcc (needed for some plugins e.g treesiter, luasnip...)
- Nerd font (i use **Iosevka Nerd Font**)
- python3
- pip
- node
- npm or yarn
- git
